# Text Autocompletion with LSTM and DistilGPT2

Автодополнение текста с использованием рекуррентной сети (LSTM) и предобученного трансформера (DistilGPT2). 
Модели получают часть текста и предсказывают следующие токены. 
Для оценки качества используется ROUGE, а также выводятся примеры автодополнений.
